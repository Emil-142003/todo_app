# TodoApp (Starter Structure)

This is a basic starter structure for a Todo app.

## Current setup

- `index.html`: main app shell.
- `src/styles.css`: base styles for layout and todo list.
- `src/app.js`: simple in-memory add/toggle/delete todo behavior.

## Run locally

Use any static server, for example:

```bash
cd todoapp
python3 -m http.server 8000
```

Then open <http://localhost:8000>.
